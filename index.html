Predição de vendas

A rede Rossmann opera cerca de 3 mil lojas em 7 países europeus. A demanda fictícia em questão é a realizar a projeção de vendas para as próximas 6 semanas para diferentes lojas da rede. Diferentes fatores podem influenciar as vendas de cada loja. Dessa forma a meta desse projeto foi utilizar essas variáveis como preditoras para a projeção de vendas por 6 semanas.

  • Python, Numpy, Pandas, Matplotlib e Seaborn
  • Jupyter Notebook, Spyder
  • Linear Regression, Lasso, Randon Forest, XGBoost

    Detecção de Fraudes em Transações Financeiras

    A demanda deste projeto foi realizar a deteção de fraudes em transações bancárias. Este tipo de projeto apresenta algumas características desafiadoras, incluindo o desbalanço entre as classes (menos de 1% do total de transações bancárias são fraudes), e também o alto volume de dados (cerca de 6 milhões de linhas no dataset). Além de desenvolver um modelo capaz de detectar fraudes nas transações bancárias ainda foi possível ao final do projeto disponibilizar predições para novas transações via requisição em API. Dessa forma, detectando assertivamente atividades fraudulentas é possível obter um retorno finaceiro maior para a empresa que oferece o serviço; segurança para os clientes e minimização de perdas por parte de indenizações.

    As principais ferramentas utilizadas foram:

  • Git, Github
  • Pyenv virtualenv
  • Python, Numpy, Pandas, Matplotlib e Seaborn
  • Jupyter Notebook
  • Logistic Regression, K-Nearest Neighbors, Randon Forest, XGBoost
